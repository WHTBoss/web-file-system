# 文件系统前端项目

## dist目录
```
该目录可以直接观看前端项目
```

### Linux安装项目（docker-compose）
```
1.安装docker与docker-compose
2.开放端口80
   firewall-cmd --zone=public --add-port=80/tcp --permanent
   firewall-cmd --reload
3.下载FileSystem.tar.gz文件并解压缩
4.进入FileSystem目录
5.运行docker-compose up命令
6.浏览器输入ip地址查看项目
```
